# Open-Source Launch Package

This folder contains launch materials for making `inference-sim` public.

- `open-source-engine-blog-draft.md`: long-form blog draft about why the
  simulator exists and what it models.
- `open-source-launch-checklist.md`: practical readiness checklist before a
  public release.
- `announcement-copy.md`: short copy for GitHub, X/LinkedIn, Discord/Slack,
  and the repository description.

This package is intentionally separate from `clean-experiments/blog-draft.md`,
which is the research idea post for predictive KV warming.
